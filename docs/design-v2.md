# 优雅弹窗 · V2 设计需求

## 1. 概述

在 V1 系统原生 Toast 通知的基础上，增加第二种通知模式——优雅弹窗。用户可通过 skill 命令在两种模式间切换。

## 2. 双模式架构

```
原始弹窗（V1）           ← 系统 Windows Toast / macOS 通知中心
优雅弹窗（V2）           ← 自定义 UI 弹窗（PowerShell + Windows Forms）
```

- 两种模式共存
- 默认使用原始弹窗（兼容现有用户）
- 通过技能命令切换，互不影响

## 3. 模式切换命令

```
- 切换到系统通知 → 设置模式为 raw
- 切换到优雅弹窗 → 设置模式为 elegant
- 当前通知模式 → 查询当前模式
```

切换命令写入 `~/.claude/notify-mode` 文件（raw / elegant），脚本启动时读取此文件决定使用哪种方式弹窗。

## 4. 优雅弹窗功能要求

### 4.1 基础功能（与 V1 一致）

| 功能 | 说明 |
|------|------|
| 项目名 | 从 transcript 读取 cwd 提取 |
| 会话名 | 第一条用户消息前 5 个字 |
| 任务内容 | 最后一条用户消息，截取 60 字 |
| 点击事件 | 点击弹窗可聚焦/处理 |
| 自动消失 | 可配置时长（2-15s） |
| 开关控制 | 支持 `~/.claude/.notifymute` 文件检测 |

### 4.2 视觉效果

| 特性 | 说明 |
|------|------|
| 无边框窗口 | 自绘窗口，无标题栏/边框 |
| 圆角 | 可配置（4-32px） |
| 透明度 | 可配置（50%-100%） |
| 阴影/发光 | 可配置外发光强度 |
| 动画 | 渐入（slideUp + fadeIn），渐出 |
| 定位 | 屏幕右下角，任务栏上方 |

### 4.3 主题系统

| 主题 | 风格描述 |
|------|---------|
| 深色简洁 | 暗色背景，白色文字，简洁线条 |
| 二次元粉紫 | 紫色渐变背景，花瓣装饰元素 |
| 萌系像素 | 粉色可爱风，圆润柔和 |
| 和风 | 米色日式简约，木质色调 |
| 赛博朋克 | 深色背景，霓虹绿紫配色 |
| Mac 毛玻璃 | 半透明毛玻璃效果 |

### 4.4 定制参数

| 参数 | 范围 | 默认值 |
|------|------|--------|
| 圆角大小 | 4-32px | 16px |
| 不透明度 | 50%-100% | 100% |
| 显示时长 | 2-15s | 5s |
| 边框发光 | 0-20px | 0px |
| 主题切换 | 6 种主题 | 深色简洁 |

## 5. 与 V1 保持一致的逻辑

- 同一套通知脚本入口（notify.ps1）
- 同一套数据提取逻辑（transcript → 项目名/会话名/内容）
- 同一套开关机制（notifymute 文件）
- 同一套 toggle 开关脚本
- macOS 优雅弹窗推迟到后续版本

## 6. 技术方案

| 组件 | 技术 |
|------|------|
| 弹窗窗口 | PowerShell + Windows Forms（无边框 Form） |
| 绘图 | System.Drawing（GDI+ 自绘） |
| 动画 | 定时器控制位置/透明度渐变 |
| 主题数据 | 配置文件定义颜色/字体/尺寸 |
| 模式切换 | 文件标记 `~/.claude/notify-mode` |
| 部署 | 嵌入 SKILL.md，AI 安装时写入 |

## 7. 文件结构

```
~/.claude/
├── notify.ps1           ← 入口脚本（读取 mode 决定弹窗方式）
├── notify-elegant.ps1   ← 优雅弹窗脚本（新增）
├── notify-toggle.ps1    ← 开关脚本
├── notify-toggle.bat    ← 开关快捷方式
├── .notifymute          ← 静默标记文件
├── notify-mode          ← 模式标记文件（raw / elegant）
└── themes/
    ├── default.json     ← 深色简洁
    ├── anime.json       ← 二次元粉紫
    ├── kawaii.json      ← 萌系像素
    ├── wa.json          ← 和风
    ├── cyber.json       ← 赛博朋克
    └── mac.json         ← Mac 毛玻璃
```
